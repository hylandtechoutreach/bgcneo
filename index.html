Introduction

Hyland + Boys & Girls Club

This repository contains material for Hyland's Partnership with Boys & Girls Club of Northeast Ohio.

Basic Info

Students will meet on Mondays and Wednesdays from 10am-12pm ET on Zoom. The first two weeks will be introductory sessions and one-off activities. The following four weeks will cover HTML & CSS, culminating in a final capstone project. All lesson plans are within this GitHub repository. Student-facing content is surfaced through this Gitbook. Students will use Replit to write code and host their websites.
